# PRD 11 — Onboarding — Zombie Waves

---

## Tutoriel (manche 1)

Le tutoriel est **intégré dans la manche 1** — pas de tutoriel séparé.

### Séquence d'onboarding

| Étape | Ce qui se passe | Ce que le joueur apprend |
|-------|----------------|------------------------|
| 1 | Spawn en Zone 1, pistolet automatique | Il a une arme, il peut tirer |
| 2 | 8 zombies lents arrivent (manche 1 facile) | Tirer sur les zombies = les tuer |
| 3 | "+$10" pop au-dessus des zombies tués | Tuer = gagner de l'argent |
| 4 | Texte "Approchez-vous du mur pour acheter une arme" | Les wall buys existent |
| 5 | Manche 1 terminée — bonus aléatoire | Bonus entre les manches |
| 6 | Manche 2 : plus de zombies | La difficulté augmente |
| 7 | Texte "Ouvrez la porte pour accéder à la prochaine zone" (quand assez d'$) | Les portes existent |

### Principes d'onboarding

- **Pas de popup bloquant** — tout est intégré au gameplay
- **Apprendre en faisant** — le joueur découvre par l'action
- **Feedback immédiat** — chaque action a une réponse visible
- **Manche 1 = tutoriel** — facile mais pas ennuyante

---

## Premier contact (nouveaux joueurs)

| Moment | Durée | Objectif |
|--------|-------|---------|
| Lobby (10s) | Le joueur voit les classes, le leaderboard, le shop — il appuie sur JOUER | Orientation rapide |
| Manche 1 (60s) | 8 zombies faciles, premiers kills, premier argent | Comprendre le core loop |
| Manche 2 (90s) | Plus de zombies, premier achat | Comprendre l'économie |
| Manche 3-4 (2 min) | Première porte, nouvelle zone | Comprendre l'exploration |
| Manche 5 (3 min) | Premier boss, zombies spéciaux | Moment "wow" |

---

## Questions clés

- [x] Un joueur comprend-il le jeu sans tutoriel formel ? → Oui, en 2 manches
- [x] Le joueur est-il "accroché" dans les 5 premières minutes ? → Oui, boss manche 5
- [x] L'onboarding est-il ennuyant pour les joueurs qui reviennent ? → Non, pas de popup
